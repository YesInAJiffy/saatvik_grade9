# ðŸ“˜ **Logarithm Rules Table**

| **Rule Name**                                  | **Algebraic Form**                                       | **Description**                           |
| ---------------------------------------------- | -------------------------------------------------------- | ----------------------------------------- |
| **Product Rule**                               | $$\log_b(xy) = \log_b x + \log_b y$$                     | Log of a product becomes a sum            |
| **Quotient Rule**                              | $$\log_b\left(\frac{x}{y}\right) = \log_b x - \log_b y$$ | Log of a quotient becomes a difference    |
| **Power Rule**                                 | $$\log_b(x^n) = n \log_b x$$                             | Exponent can be pulled in front           |
| **Root Rule** (special case of power rule)     | $$\log_b(\sqrt[n]{x}) = \frac{1}{n}\log_b x$$            | Roots become fractions                    |
| **Change of Base Formula**                     | $$\log_b a = \frac{\log_c a}{\log_c b}$$                 | Converts any log base to a new base $$c$$ |
| **Identity Rule**                              | $$\log_b(b)=1$$                                          | Because $$b^1 = b$$                       |
| **Log of 1**                                   | $$\log_b(1)=0$$                                          | Because $$b^0 =1$$                        |
| **Inverse Rule**                               | $$b^{\log_b x} = x$$                                     | Exponential and log cancel                |
| **Log of an Exponential**                      | $$\log_b(b^x)=x$$                                        | Opposite of the rule above                |
| **Change of Base to Natural Log (common use)** | $$\log_b x=\frac{\ln x}{\ln b}$$                         | Frequently used on calculators            |
| **Change of Base to Common Log (base 10)**     | $$\log_b x= \frac{\log x}{\log b}$$                      | Common log method                         |
